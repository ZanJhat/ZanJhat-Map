How to add custom block color for ZanJhat.Map

1. Create a file with extension ".bpd"
   Example: MyBlockColors.bpd

2. Add the following structure:

<Blocks>
    <Block Name="DirtBlock" NeedChange="false" Color="132, 106, 58" />
</Blocks>

Attributes

Name
    The block class name (example: DirtBlock, GrassBlock).

Color
    RGB color used on the map.
    Format: R,G,B
    Example: 132,106,58

NeedChange
    If true, the color can be modified by the environment
    (grass color map, water color map, leaves color, or custom handlers registered by mods).


Example

<Blocks>
    <Block Name="DirtBlock" NeedChange="false" Color="132, 106, 58" />
    <Block Name="GrassBlock" NeedChange="true" Color="171, 171, 171" />
    <Block Name="OakWoodBlock" NeedChange="false" Color="198, 154, 83" />
</Blocks>
