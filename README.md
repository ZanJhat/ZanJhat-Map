# ZanJhat Map

Adds a minimap and world map with customizable markers to help players navigate and track important locations.

---

## Features
- Minimap system
- World map support
- Custom markers
- Block color customization (.bpd support)
- Integration with ZanJhat Core systems

---

## Requirements
- ZanJhat Core >= 1.0.0

---

## Compatibility
- API Version: >=1.9.0.2
- SC Version: 2.4.0.0

---

## Installation
1. Install ZanJhat Core first
2. Install ZanJhat Map
3. Ensure correct load order

---

## Block Color System
Supports `.bpd` files for custom block colors in map rendering.
